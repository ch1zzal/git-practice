# Практична робота: Використання Git і GitHub

**Виконав:** Гуменюк Дмитро

## Опис виконаної роботи
У ході практичної роботи було освоєно базові команди системи контролю версій Git, принципи роботи з гілками за методологією Git Flow, а також синхронізацію локального репозиторію з віддаленим на GitHub.

### Використані команди:
* `git init` — ініціалізація локального репозиторію.
* `git branch` — керування гілками.
* `git checkout` — переключення між гілками та коммітами.
* `git merge` — злиття гілок.
* `git push` / `git pull` — синхронізація з GitHub.
